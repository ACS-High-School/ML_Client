## Features
### Client를 위한 연합학습 simulation 구축
- 연합학습을 위한 Python Package 생성 및 배포
- train data : 약 17만 개의 은행 고객 이탈 데이터
- test data : 약 11만 개의 은행 고객 이탈 데이터
- train model : Tensorflow를 활용하여 MLP 구현
- prediction : 고객이 계좌를 해지할 확률
